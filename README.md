## Export selected holidays (.cal)

- Mark your vacation days in the calendar.
- Click **Auswahl als .cal exportieren** (left of **Alle Urlaubstage l√∂schen**).
- The app downloads `holiday-planner-<year>.cal` with all selected days of the currently shown year.

## Import selected holidays (.cal)

- Click **Aus .cal importieren** (right of **Auswahl als .cal exportieren**).
- Choose a `.cal` file containing calendar events.
- The app imports event days into your saved selections (duplicates are ignored) and updates the current year view.

## Pull Request Previews

When you open a pull request, a preview of your changes will automatically be deployed to a temporary URL. This makes it easy to review and test changes before merging. The preview URL will be posted as a comment on your PR and will look like:

`https://d4koon.github.io/Holiday-Planner/pr-<number>/`

The preview updates automatically with new commits and is cleaned up when the PR is closed.
